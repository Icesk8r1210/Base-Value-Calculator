Base Value Calculator

Overview
The Base Value Calculator is an interactive web app for figure skaters to calculate the base score of a program based on chosen elements. Users can select jumps, spins, step sequences, program type, and level, and the app computes the total base value using official scoring rules.

Features

- Select program type (Short/Freeskate) and level (Juvenile → Senior)

- Add jumps, spins, step sequences, and choreographic elements

- Calculate total base value instantly

- Intuitive, user-friendly interface
